## Как использовать

1. Запустите seed скрипт:
   ```bash
   cd backend
   source venv/bin/activate
   python3 seed_db.py
   ```

2. Войдите в систему с любым из этих аккаунтов через веб-интерфейс.

3. Для регистрации нового администратора используйте пароль: `hilexahlxa123`

## Примечания

- Все пароли простые для тестирования (в продакшене используйте сложные пароли)
- Пользователи создаются только если их еще нет в базе
- Скрипт можно запускать многократно - он не создаст дубликаты
